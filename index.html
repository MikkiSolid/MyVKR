У страницы всегда есть элементы html, head, body

Блочные элементы - занимают всё пространство по ширине, поэтому всегда переносятся на новую строку

Инлайн элементы - занимают столько места, сколько требует содержимое. Если в строке хватает места, то они размещаются на одной строке

Чаще используют следующие элементы:

<div></div>- Блочный элемент

<span></span>- Инлайн элемент

<h1></h1> <h2></h2> <h3></h3> <h4></h4> <h5></h5>- Заголовки, блочные элементы

<img src="ссылка на изображение"/>- Изображение, инлайн элемент

<lable for="input-id"></lable>- Текст подпись для элементов input. При клике на этот текст фокусируются сами инпуты.

<input type="text"></input>- Строка ввода текста, якляется инлайн элементом

<input type="radio"></input>- Радио кнопка, является инлайн элементом

<input type="checkbox"></input>- Чекбокс, является инлайн элементом

<a href="Ссылка"></a>- Ссылка, инлайн элемент

<button></button>- Кнопка, инлайн элемент

<ul></ul> <ol></ol> <li></li>- Список с маркерами, нумерованный список, элемент списка. Блочные элементы

<table></table> <thread></thread> <tbody></tbody> <tr></tr> <td></td>- Таблица, заголовок таблицы, тело таблицы, строка и ячейка. Ячейка вкладываются в строки, а строки вкладываются в заголовок и тело таблицы

<form></form>- Форма. Служит для отправки данных на север. Включает в себя элементы input для ввода данных и кнопку button для отправки данных на сервер

<link rel="stylesheet" href="url"/>- Присоединение к HTML файлу других документов, обычно это файлы CSS стилей

<script src="url">код JS для выполнения</script>- Присоединение к HTML файлу JS скриптов или выполнение JS кода

<p></p>- Элемент, представляющий собой абзац текста
